# Welcome to FictOS Sandbox

This is your personal browser-based Linux environment.
Everything runs in memory — no files are saved to any server.

## Getting started

```bash
ls ~/
cat ~/README.md
nano ~/notes.txt
```

## Available apps

- **Terminal** — full bash-like shell with git support
- **Text Editor** — GUI editor backed by the same filesystem
- **Browser** — sandboxed in-world browser (MDN, Wikipedia)
- **Email** — check your welcome messages

Open the Email app to read your welcome message.
