# Functions

> **Sub-unit Summary:** A function is a named, reusable block of code. Functions are the fundamental unit of code organisation in JavaScript — everything from event handlers to API calls is written as a function. This sub-unit covers both syntax forms, how scope works, and the powerful concept of closures.

## Learning Objectives

By the end of this sub-unit, you will be able to:

- Write function declarations with parameters and return values
- Explain how hoisting applies to function declarations
- Write arrow functions with both explicit and implicit return
- Explain the difference in `this` behaviour between regular and arrow functions
- Distinguish global, function, and block scope
- Explain what a closure is and show a simple practical example
