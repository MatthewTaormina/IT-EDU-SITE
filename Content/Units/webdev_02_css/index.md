# CSS

> **Unit Summary:** CSS (Cascading Style Sheets) is the language that transforms structural HTML into the visual, interactive experiences users actually see. It is more than a styling tool — it is a layout engine, a design system, and a responsive behaviour framework. This unit is divided into six focused sub-units, each covering a distinct domain of CSS knowledge.

## Learning Objectives

By the end of this unit, you will be able to:

- Explain how CSS is linked to HTML and applied by the browser
- Write correct CSS syntax: selectors, declarations, properties, values
- Understand the cascade, specificity, and inheritance
- Target elements precisely using type, class, ID, combinator, pseudo-class, and pseudo-element selectors
- Understand and apply the box model: content, padding, border, margin
- Build one-dimensional layouts with Flexbox
- Build two-dimensional layouts with CSS Grid
- Apply typography, colour, backgrounds, and CSS custom properties
- Write responsive layouts using media queries, fluid units, and mobile-first strategy

## Prerequisites

- **Unit 01 — HTML** — you must be able to write valid, semantic HTML before you style it. CSS selectors target DOM nodes; if you don't understand the DOM, selectors won't make sense.

## How to Use This Unit

Work through sub-units **in order**. Each builds on the previous:

- You need Foundations before anything else
- Selectors target the elements you understand from Foundations
- The Box Model is required context for Layout
- Layout assumes you can already write selectors and understand the box model
- Typography & Colour can be learned alongside Layout
- Responsive Design requires all of the above
