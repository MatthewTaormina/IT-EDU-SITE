# CSS Foundations

> **Sub-unit Summary:** Before you can style anything, you need to understand *how* CSS is connected to HTML, what the syntax of a rule looks like, and how the browser decides which rule wins when two rules conflict. These three topics are the foundation everything else builds on.

## Learning Objectives

By the end of this sub-unit, you will be able to:

- Explain the three ways to link CSS to an HTML document and when to use each
- Write syntactically correct CSS rules with selectors, properties, and values
- Explain what the cascade is and how conflicting rules are resolved
- Distinguish inherited properties from non-inherited ones
