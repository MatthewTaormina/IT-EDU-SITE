# Data Structures

> **Sub-unit Summary:** Variables hold single values. Data structures hold collections. JavaScript has two built-in data structures that every developer uses daily: **arrays** (ordered lists) and **objects** (key-value maps). JSON ties them together as the universal format for transmitting data across the web.

## Learning Objectives

By the end of this sub-unit, you will be able to:

- Create, index, and mutate arrays using common methods (`push`, `pop`, `map`, `filter`, `find`, `reduce`)
- Create objects with property access using both dot and bracket notation
- Destructure arrays and objects to extract values cleanly
- Parse and stringify JSON data
- Explain when you'd reach for an array vs. an object
