# JS Foundations

> **Sub-unit Summary:** Before you can write interactive programs, you need three things: a way to run JavaScript in the browser, a way to store values, and a way to compute new values from them. This sub-unit covers all three.

## Learning Objectives

By the end of this sub-unit, you will be able to:

- Attach a JavaScript file to an HTML document correctly and explain what `defer` does
- Use the browser DevTools console to run JavaScript and inspect values
- Declare variables with `let` and `const` and explain when to use each
- Name and describe the five JavaScript primitive types
- Build expressions using arithmetic, comparison, logical, and assignment operators
- Construct dynamic strings using template literals
