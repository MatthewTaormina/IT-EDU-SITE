# HTML

> **Unit Summary:** HTML is the structural language of the web. Every web page — regardless of how complex its JavaScript framework or how polished its design — begins with an HTML document that tells the browser what content exists and what it means. This unit takes you from the document skeleton through text, links, images, lists, tables, forms, and semantic structure, with the DOM always visible as the underlying model.

## Learning Objectives

By the end of this unit, you will be able to:

- Write a valid HTML document from scratch
- Use the DOM as your mental model for how the browser reads HTML
- Apply semantic elements correctly to communicate meaning, not just appearance
- Mark up text, links, images, lists, and tables
- Build accessible, usable HTML forms
- Configure the `<head>` element for SEO and correct browser behaviour
- Distinguish structural elements (`div`/`span`) from semantic elements and know when to use each

## Prerequisites

- **Unit 00 — Web Foundations** — you need to understand what a browser does with an HTML file, what the DOM is, and how a page load works before writing your first tag.
