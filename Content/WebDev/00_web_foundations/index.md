# Web Foundations

> **Unit Summary:** Before writing a single line of code, you need to understand the system your code runs inside. This unit explains how the web works — from physical cables to the pixels on your screen.

## Learning Objectives

By the end of this unit, you will be able to:

- Distinguish the Internet from the World Wide Web
- Explain what a web browser and web server each do
- Describe how a URL addresses a resource
- Explain the HTTP request-response cycle at a high level
- Identify the roles of HTML, CSS, and JavaScript without writing any code
- Trace the complete journey of a URL from input to rendered page

## Prerequisites

None. This is the starting point.
