# Introduction to JavaScript

> **Lesson Summary:** JavaScript is the programming language of the Web. While HTML defines the content and CSS defines the presentation, JavaScript specifies the behavior. In this lesson, we will see how to attach a script to an HTML document and the absolute basics of browser execution.

## Linking JavaScript to HTML

Just like CSS, you can write JavaScript directly in an HTML file or link it as an external file. For modularity and best practices, we always use external files.

```html
<!-- At the end of the <body> tag -->
<script src="app.js"></script>
```
