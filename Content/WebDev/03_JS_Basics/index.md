# JavaScript Basics

Welcome to **JavaScript Basics**. While HTML provides the structure and CSS creates the design, JavaScript (JS) brings your websites to life. By the end of this unit, you will be able to write functional code that reacts to user input and dynamically alters the DOM.

This unit focuses on programmatic logic inside the browser environment, preparing you for DOM manipulation, algorithmic problem solving, and modern frontend frameworks like React.

## Path Overview

This unit is divided into the following milestones:

- **Foundations:** Learn how to declare variables, understand data types (Strings, Numbers, Booleans), and implement basic operators.
- **Control Flow:** Dictate the path of your script using `if/else` logic and iterate over repetitive tasks with loops.
- **Functions:** Encapsulate logic into reusable blocks, understand parameters, return statements, and the modern Arrow function syntax.
- **Data Structures:** Store and retrieve complex data using Arrays and Object literals.
- **The DOM:** Connect your JavaScript directly to your HTML and CSS to create interactive experiences.
