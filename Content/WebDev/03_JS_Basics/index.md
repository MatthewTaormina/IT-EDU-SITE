# JavaScript Basics

> **Unit Summary:** HTML defines the structure of a web page. CSS defines its appearance. JavaScript defines its *behaviour*. In this unit you will progress from writing your very first script to selecting and manipulating DOM nodes, handling user events, and organising data with arrays and objects.

While HTML and CSS are *declarative* (you describe what you want), JavaScript is an *imperative, general-purpose programming language* executed inside the browser. Every interactive element on the modern web — dropdown menus, form validation, live search, animations triggered by scroll — is driven by JavaScript.

---

## Why JavaScript?

- **Only language native to the browser** — no compilation step, no plugin required
- **Full-stack capable** — the same language runs on the server with Node.js
- **Enormous ecosystem** — npm hosts millions of packages; frameworks like React and Vue are built on top of vanilla JS
- **Gateway to modern frontend** — understanding vanilla JS first makes every framework easier to learn
