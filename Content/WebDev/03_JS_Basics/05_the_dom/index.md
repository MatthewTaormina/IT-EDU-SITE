# The DOM

> **Sub-unit Summary:** Everything you have learned so far — variables, functions, arrays, objects — comes together when you connect JavaScript to the browser. The Document Object Model (DOM) is the programmatic interface to the live HTML on the page. Through the DOM you can read, create, update, and delete elements and attributes, and respond to user interactions via events.

## Learning Objectives

By the end of this sub-unit, you will be able to:

- Select single and multiple DOM elements using `querySelector`, `querySelectorAll`, and `getElementById`
- Read and write element content (`textContent`, `innerHTML`)
- Add, remove, and toggle CSS classes and modify attributes
- Create new elements and insert or remove them from the DOM
- Register event listeners with `addEventListener` and handle common events
- Access the event object and use `preventDefault()`
