# Selectors & Specificity

> **Sub-unit Summary:** The selector is the most expressive part of a CSS rule. It determines which DOM nodes a rule targets. Mastering the full selector system — from the simplest type selector to complex pseudo-class chains — gives you surgical precision over any element on the page.

## Learning Objectives

By the end of this sub-unit, you will be able to:

- Use type, class, and ID selectors correctly
- Write combinator selectors to target elements by their DOM relationships
- Apply pseudo-classes to style elements based on state or position
- Use pseudo-elements to target specific parts of an element
- Calculate specificity scores and predict which rule wins
