# Grid

> **Sub-unit Summary:** CSS Grid is a two-dimensional layout system — it controls rows and columns at the same time. Where Flexbox is item-driven (items decide how they're sized), Grid is container-driven (the container defines the structure and items slot into it). Grid is the right tool for page-level layout and any design that requires precise two-dimensional placement.
