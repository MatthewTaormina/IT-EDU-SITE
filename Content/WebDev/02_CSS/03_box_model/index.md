# The Box Model

> **Sub-unit Summary:** Every element on a web page is a rectangular box. It doesn't matter if it looks like a circle, a triangle, or an irregular shape — at its core, CSS treats everything as a box. Understanding those four layers (content, padding, border, margin) and how they affect sizing is the mental model behind every layout decision you will ever make.

## Learning Objectives

By the end of this sub-unit, you will be able to:

- Identify the four layers of the box model: content, padding, border, margin
- Explain the difference between `content-box` and `border-box` sizing — and why `border-box` is the universal standard
- Understand display types (`block`, `inline`, `inline-block`, `none`) and normal document flow
