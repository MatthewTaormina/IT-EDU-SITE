# Front-End Intermediate (web)

> **Course Summary:** This course fills the critical gap between "I can build a static page" and "I can work like a professional developer." You will master Git, the terminal, asynchronous JavaScript, modern ES6+ syntax, browser storage APIs, CSS animations, and the full power of browser DevTools.

## Course Objectives

By the end of this course, you will be able to:

- Use Git and the command line to manage code, collaborate, and deploy via GitHub Pages
- Write asynchronous JavaScript using callbacks, Promises, and async/await
- Consume REST APIs using the Fetch API and render the data in the DOM
- Use ES6+ features — destructuring, spread/rest, modules, optional chaining — in real code
- Persist data client-side using `localStorage` and navigate history with the History API
- Animate UI elements with CSS transitions, transforms, and `@keyframes`
- Debug HTML, CSS, JavaScript, and network requests using browser DevTools

## Prerequisites

- **[Front-End Basics (web)](../../Courses/webdev/index.md)** — you must be comfortable with HTML, CSS, and vanilla JavaScript (DOM manipulation, events) before starting here.

## How to Use This Course

- **Work in order.** Git & CLI is Unit 1 because you will use Git in every subsequent unit. Do not skip it.
- **Use DevTools constantly.** Even before you reach the DevTools unit, open your browser developer tools on every page you visit.
- **Build the capstone as you go.** The [Intermediate Capstone Project](../../Projects/webdev_intermediate_capstone/index.md) builds incrementally — you can start the skeleton after Unit 2 and add features as you complete each unit.
- **Type every code example.** Passive reading creates an illusion of understanding. Active typing creates muscle memory.
