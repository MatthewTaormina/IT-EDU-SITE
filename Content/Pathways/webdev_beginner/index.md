# Learning Pathway: Web Developer — Zero to Employable

> **Who this is for:** Complete beginners with no prior programming or IT experience.
> **Goal:** Build a real portfolio website from scratch and understand every line of code in it.
> **Estimated Time:** ~60 hours

---

## Pathway Overview

This pathway takes you through the Front-End Basics (web) course in a single opinionated sequence. Every step builds directly on the last. Do not skip.

| Step | Content | Type | Skills Gained |
| :--- | :--- | :--- | :--- |
| 1 | [Web Foundations](../../Units/webdev_00_web_foundations/index.md) | Unit | How the web works: browsers, servers, DNS, HTTP |
| 2 | [HTML](../../Units/webdev_01_html/index.md) | Unit | Document structure, semantic HTML, accessibility |
| 3 | [CSS](../../Units/webdev_02_css/index.md) | Unit | Styling, layout (Flexbox, Grid), responsive design |
| 4 | [JavaScript Basics](../../Units/webdev_03_js_basics/index.md) | Unit | Variables, functions, DOM, events |
| 5 | [Capstone — Personal Portfolio](../../Projects/webdev_capstone_portfolio/index.md) | Project | Apply all four units to a real deliverable |

---

## Milestones

**Milestone 1 — "I understand how the web works"**
After Unit 1, you should be able to explain what happens when you type a URL into a browser and press Enter, without looking anything up.

**Milestone 2 — "I can build a static page"**
After Units 2 and 3, you should be able to build a complete, styled, responsive HTML/CSS page from a design specification.

**Milestone 3 — "I can make it interactive and ship it"**
After Unit 4 and the capstone project, you should have a deployed portfolio website with interactive JavaScript features and be able to walk through every line of code in it.

---

## Recommended Tools & Environment

Before starting, set up:

- A modern browser (Chrome or Firefox) with DevTools — you will use DevTools constantly
- [Visual Studio Code](https://code.visualstudio.com/) with the **Live Server** extension
- A free [GitHub](https://github.com) account for version control and deploying your portfolio
- A free [Netlify](https://netlify.com) or [GitHub Pages](https://pages.github.com) account for hosting

None of these require payment. All are industry-standard tools used by professional developers.
