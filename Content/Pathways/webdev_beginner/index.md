# Learning Pathway: Web Developer — Zero to Employable

> **Who this is for:** Complete beginners with no prior programming or IT experience.
> **Goal:** Go from zero to a deployed portfolio and a professional developer workflow.
> **Estimated Time:** ~105 hours (two courses + two capstone projects)

This pathway combines **two full courses** into one end-to-end journey. Course 1 teaches you to build pages; Course 2 teaches you to work like a professional. Do both in order — each builds directly on the last.

## Milestones

**Milestone 1 — "I understand how the web works"**
After Course 1, Unit 1: explain what happens when you type a URL and press Enter, from DNS lookup to pixels on screen, without looking anything up.

**Milestone 2 — "I can build a styled, interactive page"**
After Course 1, Units 2–4: build a complete, responsive HTML/CSS/JS page from a design spec — no tutorial to follow.

**Milestone 3 — "I have a deployed portfolio"**
After Step 2 (Capstone 1): a live, publicly accessible portfolio site hosted on Netlify or GitHub Pages that you built and understand line-by-line.

**Milestone 4 — "I work with Git and the terminal"**
After Course 2, Unit 1: initialise repos, commit, branch, open pull requests, and deploy via GitHub Pages — all from the command line, no GUI needed.

**Milestone 5 — "I write modern, asynchronous JavaScript"**
After Course 2, Units 2–3: consume a public REST API with `fetch`, render its data in the DOM, and write that code using ES6+ syntax (destructuring, modules, async/await) as a default habit.

**Milestone 6 — "I can debug and ship production-quality work"**
After Course 2, Units 4–6 + Capstone 2: use `localStorage`, CSS animations, and DevTools to build, inspect, and ship an API-powered interactive app — version-controlled on GitHub.

## Recommended Tools & Environment

Set up once before starting Step 1 — you will use all of these throughout both courses:

- A modern browser (Chrome or Firefox) with DevTools
- [Visual Studio Code](https://code.visualstudio.com/) with the **Live Server** extension
- [Git](https://git-scm.com/) installed and configured (`git config --global user.name/email`) — needed for Course 2 Unit 1
- A free [GitHub](https://github.com) account
- A free [Netlify](https://netlify.com) or [GitHub Pages](https://pages.github.com) account for hosting

None of these require payment. All are industry-standard tools used by professional developers.
